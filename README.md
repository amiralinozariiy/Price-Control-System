# Price Control System

## 📌 Overview

This project analyzes product prices using Microsoft Excel and Python.

The goal is to compare prices, identify changes, and generate reports.

---

## Features

- Import Excel files
- Compare previous and current prices
- Calculate price differences
- Generate reports

---

## Technologies

- Python
- Microsoft Excel
- Pandas

---
